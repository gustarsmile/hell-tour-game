## 文字版網址規則

- 章回索引頁：`https://www.taolibrary.com/category/category48/c48002b.htm`（GB 系編碼，需以 `TextDecoder('gb18030')` 或等效方式解碼，直接以 UTF-8 讀取會亂碼）
- 單回內文頁：`https://www.taolibrary.com/category/category48/c48002b/{N}.htm`（頁面本身為 UTF-8 帶 BOM，可直接讀取；`N<10` 需補零，如 `06.htm`）
- **回數與頁碼對應規則**：`第 K 回` 對應頁碼 `N = K + 2`（頁 01 為〈玉詔〉、頁 02 為〈序〉，頁 03 起為正文第一回）。已用頁面內 `<h3>第 K 回 ……</h3>` 標題逐一核對，規則成立，無例外。
- 索引頁共列 **65 個頁面連結**，其中正文共 **62 回**：頁 01〈玉詔〉、頁 02〈序〉、頁 65〈跋〉非正文回目；頁 03～頁 64 對應第 1～62 回（頁 64 之 `<h3>` 為「第六十二回 拜會地藏王菩薩，遊記著成開盛會」，頁 65 內文開頭為「跋」，均已實查確認）。`validateKarmaCard` 容許 chapter 1–65 為保守上限，實際引用回數最大為 58（hall10 區間末回）。

## 複校方法

1. 下載索引頁（`curl`），以 Node `TextDecoder('gb18030')` 解碼，取得全部 65 個頁面連結（含正文 62 回之回目）清單。
2. 對照表中每一回，逐一下載對應頁碼原文（`curl` + Node UTF-8 解碼，去除 BOM），核對頁內 `<h3>` 標題與 `karmaCard.result`／`sin` 是否相符。
3. 對疑似或待補回數（hall1），以關鍵字（秤、斗秤、短斤、缺兩、摻假、欺客）掃描全部 65 個內文頁（正文 62 回＋玉詔／序／跋），鎖定實際內容相符之回。
4. 每回節錄一句原文作為證據，記錄於下表。

## hall1 補回數依據說明（詳細）

- 以關鍵字「秤／斗秤／短斤／缺兩／摻假／假貨／欺客」掃描全部 65 個內文頁，全書字面上**沒有**任何一回以「秤鉤」為獄名或標題（已逐頁確認，索引回目清單亦無「秤鉤」字樣）。
- 掃描結果唯一命中「秤」相關內容的是**第二十六回〈遊斷筋剔骨小地獄〉**，內容為三名罪魂自述：菜販在秤砣灌鉛、收購廢五金者在秤錘上作弊（十斤只秤七斤）、家庭主婦沉迷賭博——前兩則供詞正是「斗秤不公、摻假欺客」的典型情節，且獄官明白告誡：「世間的生意人，千萬要憑良心買賣，勿貪小利，斗秤不公，地獄嚴刑不赦」。
- 依任務簡報 Step 3.3 之判準（「找『秤鉤小地獄』或**斗秤／短斤相關之回**」），第二十六回屬於「斗秤／短斤相關之回」，故採為 `source.chapter`。
- **「秤鉤」獄名之典籍查證（2026-07-08 補校）**：曾嘗試查證「秤鉤小地獄」是否見於《玉曆寶鈔》——實查 repay.tw《玉曆寶鈔：六道輪迴的真相》上／中／下三部完整文字版（`https://repay.tw/hell100.html`、`hell200.html`、`hell300.html`），全文僅一處出現「秤」字：罪業條目「（三）買東西要重秤，以佔人家便宜。」，**並無「秤鉤」獄名**；另查地獄變相圖之各殿小地獄名單（含第二殿黑雲沙、糞尿泥等十六小獄），亦無「秤鉤」。網路搜尋「玉曆寶鈔 秤鉤 小地獄」無任何可靠文獻佐證。既查無確證，依零捏造原則**不將「秤鉤」具名歸給《玉曆寶鈔》**，於 `karmaCard.result` 末尾補註已驗證之事實「（本回原著獄名為「斷筋剔骨」）」，未更動其他卡面遊戲敘事（testimony／mirror／judgement 選項等）。
